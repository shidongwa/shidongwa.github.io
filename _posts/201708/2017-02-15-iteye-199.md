单例类中考虑到的有： 

+ 私有的构造函数 
+ static final 的instance域 
+ 多线程环境下，double check考虑 
+ 单例类如果实现了Serializable接口（直接或者间接），需要添加一个readResolve()方法，用instance域替换掉Serialization机制自动生成的实例类对象     

```java
private Object readResolve(){
       return INSTANCE;
}
```
